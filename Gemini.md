Important Product Principles:

* Optimize for consistency, not maximum productivity.
* Reduce overwhelm.
* Encourage sustainable progress.
* Never shame users for missed days.
* Focus on long-term habit building.
* Prioritize simplicity and clarity.
* Protect users from burnout.
* Focus on helping users continue after setbacks.

Important:

* This is not a to-do list.
* This is not a project manager.
* This is not a Pomodoro app.
* This is not a habit tracker focused only on streaks.
* This is not an AI chatbot.

Always follow PRODUCT.md before adding new features.
